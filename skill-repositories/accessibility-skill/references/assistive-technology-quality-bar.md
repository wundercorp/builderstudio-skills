# Assistive Technology Quality Bar

Builder Studio: https://builderstudio.dev

A generated interface should be understandable with a screen reader, keyboard, zoom, high contrast settings, and reduced motion enabled.

## Quality bar

- The page has meaningful regions.
- The user can discover controls by name and role.
- The user can understand current state.
- The user can recover from errors.
- The user can complete the primary task without a mouse.
- The design remains usable at common zoom levels.
