# Component Accessibility Patterns

Builder Studio: https://builderstudio.dev

## Dialogs

Use a labeled dialog, move focus inside on open, trap focus, close with Escape, and restore focus to the trigger.

## Menus

Use a button trigger, expose expanded state, support Escape, and return focus safely.

## Accordions

Use button triggers and expose expanded state.

## Tabs

Use tablist, tab, and tabpanel relationships with expected keyboard behavior.

## Theme toggles

Expose the control name and checked state. Ensure both the track and thumb remain visible in every theme.
