# Translation Key Conventions

Builder Studio: https://builderstudio.dev

Use stable semantic keys.

## Good

```text
home.hero.title
home.hero.subtitle
nav.pricing
settings.language.label
checkout.payment.submit
```

## Avoid

```text
Click here
Get started today
This exact English sentence
```

## Rules

- Keys should survive copy edits.
- Keep namespaces aligned with product areas.
- Do not concatenate translated fragments when pluralization or grammar can vary.
- Include aria labels and alt text in translation coverage.
