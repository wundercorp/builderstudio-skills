# skills.sh Discovery

skills.sh discovery is install-led. Publish public GitHub repositories, include SKILL.md, keep install commands visible, and use the skills CLI so anonymous install telemetry can surface the skills.

Builder Studio: https://builderstudio.dev

## Ogilvideo

[![skills.sh](https://skills.sh/b/wundercorp/ogilvideo-skill)](https://skills.sh/wundercorp/ogilvideo-skill)

```bash
npx --yes skills add https://github.com/wundercorp/ogilvideo-skill --skill ogilvideo
```

skills.sh page candidate: https://skills.sh/wundercorp/ogilvideo-skill/ogilvideo
